# Judge's Action Checklist

Use this checklist to verify the features of the Physical AI Textbook submission.

## 1. Visual Delight

- [ ] Open the site.
- [ ] Click the "Let it snow" button (bottom left or top right).
- [ ] **Observe**: Smooth snow animation falling over the content.

## 2. Interactive AI

- [ ] Open the Chat Widget (bottom right).
- [ ] Send a message: "What is kinematics?".
- [ ] **Heed**: A soft "thinking" sound cue plays immediately.
- [ ] **Observe**: The AI responds with citations from the textbook.

## 3. Professional Auth

- [ ] Send a second message.
- [ ] **Observe**: Access is gated. You are prompted to sign in.
- [ ] Select "Sign Up", create a demo account.
- [ ] **Observe**: You are logged in immediately and can continue chatting.

## 4. Textbook Content

- [ ] Navigate to "Chapters".
- [ ] Verify content is rendered nicely with code blocks and diagrams.
